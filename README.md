# 房多多 Agent Demo

移动端优先的杭州房产服务 Agent 交互原型。首版使用模拟 MCP 数据，重点验证从房产问答到支付宝服务承接的完整用户体验。

[打开房多多 Agent Demo](https://wk455318123.github.io/homeduo-agent-demo/)

## 汇报材料

- [支付宝房产机会汇报（新房补充版）](https://wk455318123.github.io/homeduo-agent-demo/reports/alipay-housing-newhouse/)

## 演示场景

- 杭州城市房价趋势
- 300 万预算下的余杭、萧山对比
- 未来科技城 5000 元以内租房
- 租房位置、预算、租住类型的三步需求澄清
- 多平台租房供给推荐、完整频道、地图找房与预约看房
- 万科城市花园小区成交观察
- 具体小区二手房价、租金趋势与租售比综合卡片
- 保租房、人才房等政策住房资格预匹配、项目聚合与申请进度
- 政策公告结构化、开放截止提醒与支付宝消息订阅
- 审核通过后的签约、缴租、公积金提取与入住生活服务
- 学区问题的能力边界与官方渠道降级

## 服务闭环

- 聚合在售与可租房源
- 模拟聚合贝壳租房、安居客、58同城、自如、我爱我家等多家供给
- 移动端优先交互，桌面浏览器中也以手机产品形态展示
- 回答前展示意图识别、模拟 MCP 与风控组装过程，回答内容分段出现
- 未覆盖问题提供明确能力兜底，不会静默无响应或强行套用趋势结论
- 动态房贷月供计算器
- 公积金授权查询与租房提取入口
- 花呗免押、生活便利度等后续服务

## 后续接入真实 MCP

建议将当前模拟响应替换为统一适配层，保留以下返回结构：

```json
{
  "answer": "经过蒸馏的自然语言结论",
  "metrics": [],
  "visuals": [],
  "source": {
    "provider": "趋势动物",
    "as_of": "2026-06-07"
  },
  "service_actions": [],
  "risk_tags": []
}
```

趋势动物侧返回蒸馏答案与图片图表；支付宝侧负责意图识别、合规过滤、回答编排和服务推荐。

# 房多多 Agent Demo

移动端优先的杭州房产服务 Agent 交互原型。首版使用模拟 MCP 数据，重点验证从房产问答到支付宝服务承接的完整用户体验。

## 在线体验

[打开房多多 Agent Demo](https://wk455318123.github.io/homeduo-agent-demo/)

## 演示场景

- 杭州城市房价趋势
- 300 万预算下的余杭、萧山对比
- 未来科技城 5000 元以内租房
- 租房位置、预算、租住类型的三步需求澄清
- KA 合作平台房源推荐、详情与预约看房
- 万科城市花园小区成交观察
- 学区问题的能力边界与官方渠道降级

## 服务闭环

- 聚合在售与可租房源
- 模拟自如 KA 房源接入与 Agent 智能匹配
- 动态房贷月供计算器
- 公积金授权查询与租房提取入口
- 花呗免押、生活便利度等后续服务

## 后续接入真实 MCP

建议将当前模拟响应替换为统一适配层，保留以下返回结构：

```json
{
  "answer": "经过蒸馏的自然语言结论",
  "metrics": [],
  "visuals": [],
  "source": {
    "provider": "趋势动物",
    "as_of": "2026-06-07"
  },
  "service_actions": [],
  "risk_tags": []
}
```

趋势动物侧返回蒸馏答案与图片图表；支付宝侧负责意图识别、合规过滤、回答编排和服务推荐。
